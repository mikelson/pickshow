PickShow Screen Saver

What is it?

PickShow Screen Saver is a screen saver utility that you can install on your Mac OS X computer. It lets you pick a folder on your hard drive, then, when your screen saver starts, it will show you all the pictures in that folder in random order. If you like, you can choose subfolders to not show (such as the ones with embarassing photos). In addition, while the screen saver is running, you can use your keyboard to skip pictures you don't want to see, go back to ones you want to see again, move a really bad picture to the Trash, or exit the screen saver and open an interesting picture in Finder.

How do I set it up?

  1. Choose Apple menu > System Preferences, click Desktop & Screen Saver, and then click Screen Saver.
  2. In the Screen Savers list, towards the bottom, under "Other", click PickShowScreenSaver.
  3. Click Options.
    • Click the ? button to get help (this page).
    • Picture Folder: type the full folder name where your pictures are, or click Choose to browse to it.
    • Folders to Skip: Click "+" to select folders to skip. Select a folder already in the list and click "-" to remove it and show pictures from it again.
    • You can enter a number of seconds to show each picture.
  4. Click OK.

What keyboard keys control it?

key(s)action
/, hshow this help screen
→ (right arrow)next picture
← (left arrow)previous picture
fshow current picture in Finder
d, back, delete, ⌫move current picture to Trash
anything elseexit screen saver
